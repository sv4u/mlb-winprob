▶ Models explained

Logistic Regression

A regularised linear model and the interpretable baseline. All 66 features are z-score normalised before fitting, so the learned coefficients directly quantify each feature's additive contribution to the log-odds of the home team winning.

  • L2 regularisation (C = 1.0), L-BFGS solver
  • SHAP = coef × z-score, no approximation needed
  • Best for: speed, auditing, explainability

LightGBM

Microsoft's gradient-boosted tree library. Grows an ensemble of shallow trees in sequence, capturing non-linear interactions between features.

  • Optuna-tuned: learning rate, num_leaves, n_estimators
  • Tree-based SHAP explanations via TreeExplainer
  • Best for: fast batch inference, non-linear interactions

XGBoost

DMLC's gradient-boosted tree library with L1/L2 penalties on leaf weights. Typically achieves the best single-model Brier score.

  • Optuna-tuned: learning rate, max_depth, regularisation
  • Complementary error surface to LightGBM
  • Best for: highest single-model accuracy

CatBoost

Yandex's gradient-boosted tree library with ordered boosting and symmetric decision trees. Resistant to overfitting.

  • Ordered boosting reduces prediction shift
  • Symmetric trees, L2 leaf regularisation
  • Best for: robustness, low-variance ensemble partner

Neural Network (MLP)

A multi-layer perceptron with two hidden layers (128, 64 units) and ReLU activations. Provides model diversity for stacking.

  • Architecture: 128 → 64 → 1, Adam optimizer
  • L2 weight decay for regularisation
  • Best for: ensemble diversity, nonlinear interactions

Stacked Ensemble default

Takes the calibrated probability outputs of all five base models and trains a logistic-regression meta-learner to find the optimal blend.

  • Inputs: all 5 base model calibrated probabilities
  • Meta-learner: LogisticRegression (C = 0.5)
  • Best for: maximum production accuracy

▸ Training techniques
Calibration
Isotonic for tree models, Platt (sigmoid) for linear/neural — so a predicted 65% actually wins ~65% of the time.
Time-weighted training
Exponential decay (0.12/season) gives recent seasons more influence, adapting to rule changes.
Optuna HPO
60-trial Bayesian search over hyperparameters, evaluated on 3-season expanding window.
Expanding-window CV
For season N, the model trains on all prior seasons. No future data leaks. All metrics are fully out-of-sample.
