Pipeline Controls

Update Season idle

Refresh the current season's schedule, gamelogs, crosswalk, and feature matrices. Non-destructive — only overwrites the specified seasons. Auto-reloads on completion.

Full Reingest idle

Destructive: clears all processed data and re-ingests every season (2000–present) from scratch. This is a heavy operation.

Retrain Models idle

Destructive: removes all existing model artifacts, then trains all 6 production models from scratch. Auto-reloads on completion.
